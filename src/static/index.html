H5P SCORM packager

This utility allows you to package H5P files into SCORM files. It's licensed under the MIT license and the source code can be found on GitHub.

To package a H5P file into a SCORM package, select a file from your disk and click on "Package". Your browser will the download the SCORM package as a zip file. You can then put the zip file into a LMS of your choice.
Note: Your H5P file is uploaded to the server of this utility during this process!
